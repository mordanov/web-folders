# Mainpage icon prompts

Target model: **gpt-image-2**
Size: **256×256 px**, output format: PNG

## Style guide (include in every prompt)

> Soft 3D render style, slightly clay-like and puffy. Rounded square icon shape with a medium purple background (#7c5cbf). All objects are light lavender/white with soft purple shadows and gentle highlights. Warm ambient lighting from the top-left. No text, no labels. Clean, family-friendly aesthetic consistent with a home dashboard app.
